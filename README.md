# Calculadora Llaveros 3D — PWA package

Contenido del paquete:
- index.html  (aplicación web PWA)
- manifest.json
- sw.js
- icons/icon.svg
- README.md

Instrucciones rápidas (gratis):
1. Crea un repositorio público en GitHub llamado `calculadora-llaveros`.
2. Sube todos los archivos del ZIP al root del repositorio.
3. Activa GitHub Pages en `Settings -> Pages` seleccionando la rama `main` y carpeta `/ (root)`.
4. Espera ~1 minuto y abre la URL `https://<tu-usuario>.github.io/calculadora-llaveros/`.
5. En Android Chrome: `Agregar a pantalla de inicio` o el navegador te sugerirá instalar.

Nota:
- Para un service worker completamente fiable debes servir los archivos bajo HTTPS (GitHub Pages lo hace).
- PWABuilder puede convertir esa URL en un APK si deseas instalar un archivo .apk.
